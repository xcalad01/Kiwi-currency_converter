# Kiwi-currency_converter
## CLI application
### Usage
Usage: currency_converter.py [options]

Options:
  -h, --help            show this help message and exit
  
  --amount=AMOUNT
  
  --input_currency=INPUT_CURRENCY
  
  --output_currency=OUTPUT_CURRENCY
